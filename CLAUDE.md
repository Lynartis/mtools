## Project Overview

This repository contains Blender scripting projects (Python scripts for Blender's scripting API).

## Blender Scripting Conventions

- Blender Python API: `bpy` module
- Scripts target Blender's embedded Python interpreter
- Use Blender's operator system (`bpy.ops`) and data API (`bpy.data`, `bpy.context`)
